This project predicts which face of a CubeSat will be exposed to the Sun during its orbit. It helps improve mission planning by forecasting thermal stress on satellite surfaces.

Tools & Technologies:

Python

LSTM (Deep Learning model for face classification)

Operational Research algorithms

Synthetic dataset simulating orbital dynamics and temperatures

Features:

LSTM-based model to classify CubeSat faces exposed to sunlight.

Python algorithm to detect the face orientation over time.

Synthetic dataset generation for training and validation.

Predictive analysis for thermal stress and mission planning.
